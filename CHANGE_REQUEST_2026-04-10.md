# 项目变更单（计划变更二）

## 1. 基本信息
- 变更编号：CR-2026-02
- 变更名称：企业上报与市/省审核流程支持手机端操作
- 提出日期：2026-04-10
- 提出人：项目组
- 变更类型：需求变更 + 进度计划变更
- 关联项目：云南省企业就业失业数据采集系统

## 2. 变更背景
随着使用场景扩展，业务方提出在移动设备上完成关键流程的需求：
- 企业上报（备案提交、月报提交）
- 市级审核
- 省级审核与上报

现有系统以桌面端展示为主，虽有基础响应式能力，但未针对手机端操作路径进行专门优化，需要纳入项目变更管理。

## 3. 变更内容
### 3.1 功能范围
- 前端移动端增强（最小可运行改动）：
  - 增加手机端快捷流程入口（按角色展示）
  - 调整小屏触控操作体验（按钮尺寸、输入框高度、单列布局）
  - 不改变后端接口与业务流程
- 后端：
  - 本次不新增接口，仅复用现有 API

### 3.2 非目标范围
- 不新增原生 App
- 不做离线能力
- 不引入新鉴权体系

## 4. 影响分析
- 进度影响：中等
- 成本影响：低（以前端改造工时为主）
- 风险：
  - 风险1：不同手机屏幕尺寸导致局部布局错位
  - 风险2：长表单在移动端连续输入时可用性下降
  - 风险3：移动网络环境下接口响应波动
- 应对措施：
  - 采用最小可运行改造，优先保障关键链路可操作
  - 完成多分辨率抽样测试（常见 Android/iOS 视口）
  - 保持桌面端行为不变，降低回归风险

## 5. 进度计划变更

- 变更后总周期：2 个月 + 2 周

  - 2026-04-16 ~ 2026-04-20：联调与回归测试
  - 2026-04-21 ~ 2026-04-24：缺陷修复与验收准备

- 手机端可完成省级审核与上报关键操作
- 不影响原桌面端流程
